На основе предоставленного описания проекта Genome Benchmark Generator и уже реализованных модулей, следующим логическим шагом будет создание главного файла yatomat.py, который будет отвечать за генерацию FASTA и GFF3 файлов.



Я создал два файла:

1. `yatomat.py` - главный модуль программы, который:
- Принимает конфигурационный файл JSON и параметры командной строки
- Создает все необходимые объекты для генерации хромосомы
- Генерирует последовательность и аннотации
- Записывает результаты в форматах FASTA и GFF3
- Поддерживает сжатие выходных файлов
- Имеет подробное логирование
- Обрабатывает ошибки

2. `example_config.json` - пример конфигурационного файла, который показывает:
- Структуру конфигурации
- Все доступные параметры
- Типичные значения параметров

Основные особенности реализации:

1. Форматирование файлов:
- FASTA записывается с разбивкой на строки по 60 символов
- GFF3 следует спецификации формата
- Поддерживается сжатие gzip

2. Конфигурация:
- Гибкая структура JSON
- Возможность задать все параметры
- Значения по умолчанию для необязательных параметров

3. Валидация:
- Проверка входных параметров
- Создание каталогов при необходимости
- Обработка ошибок с информативными сообщениями

4. Удобство использования:
- Аргументы командной строки
- Настраиваемые префиксы файлов
- Возможность задать seed для воспроизводимости

Для использования программы:
```bash
python yatomat.py config.json --output-dir output --prefix chr1 --compress --seed 42
```
